# Vim Configuration

My personal Vim configuration files for quick setup on new machines.

## Setup

### Quick Install
```bash
git clone https://github.com/yourusername/dotfiles.git
cd dotfiles
./setup.sh
```

### Manual Install
```bash
git clone https://github.com/yourusername/dotfiles.git
ln -sf /path/to/dotfiles/.vimrc ~/.vimrc
ln -sf /path/to/dotfiles/.vim ~/.vim
```

## Update
```bash
cd /path/to/dotfiles
git pull
```

## Save Changes
```bash
cd /path/to/dotfiles
git add .
git commit -m "Update vim configuration"
git push
```
